#dataAllEmpty 数据是否为空

dataAllEmpty< T extends object> (data: T,options: DataAllEmptyOptions = {}): boolean | any[]

interface DataAllEmptyOptions { returnKeys?: boolean; parentKey?: string; checkEmptyFn?: (value: any) => boolean; ignoreKeys?: string[]; maxDepth?: number; }

  • {object}data: 需要检查的对象
  • {DataAllEmptyOptions} options
  • {boolean} options.returnKeys: 是否返回空值的路径（默认返回 true/false）
  • {string} options.parentKey: 父级路径（默认为空字符串）
  • {function} options.checkEmptyFn: 自定义判断是否为空的函数（默认使用内置函数）
  • {array< string>} options.ignoreKeys: 忽略的键名（默认为空数组）
  • {number} options.maxDepth: 最大递归深度（默认无限制）
  • {array< string> | boolean} returns: 全部为空时返回 `false`，否则返回 `true`

#dataEqual 数据对比

dataEqual(obj1: any,obj2: any,options: { returnKeys?: boolean; arrayDiff?: boolean } = {}):boolean | string[]

  • {object|array} obj1: 第一个对象/数组
  • {object|array} obj2: 第二个对象/数组
  • {object} options: 配置项

  • {boolean} options.arrayDiff: 是否检查数组内差异（默认 true）
  • {boolean | string[]} returns: 相等返回 false，不相等返回 true 或不同 key true）
