<CodeRunner code={`
const data = zztool.getUrlParam(); // 获取当前页面参数
console.log(data)
const data1 = zztool.getUrlParam('https://example.com/test?name=tom&age=18&sex=1') // string
console.log(data1)
`} />

### props
| 属性名 | 说明 | 类型 | 必填 | 默认值 |
| --- | --- | --- | --- | --- |
| *url* | *url链接地址 node环境下参数必填* | *String* | *❌* | *当前域名地址* |
{/* | *end* | *结束时间* | *String* | *✅* | *-* |
| *step* | *步长* | *String* | *❌* | *01:00* |
| *format* | *格式* | *String* | *❌* | *hh:mm* | */}

### 返回值
格式化参数
