<CodeRunner code={`
const data = zztool.getTimeStep('08:00', '10:00', '01:00', 'hh:mm'); // ["08:00", "09:00", "10:00"]
console.log(data)
const data1 = zztool.getTimeStep('08:00', '10:00', '01:00', 'hh:mm:ss'); // ["08:00:00", "09:00:00", "10:00:00"]
console.log(data1)
`} />

### props
| 属性名 | 说明 | 类型 | 必填 | 默认值 |
| --- | --- | --- | --- | --- |
| *start* | *开始时间* | *String* | *✅* | *-* |
| *end* | *结束时间* | *String* | *✅* | *-* |
| *step* | *步长* | *String* | *❌* | *01:00* |
| *format* | *格式* | *String* | *❌* | *hh:mm* |

### 返回值
返回值
