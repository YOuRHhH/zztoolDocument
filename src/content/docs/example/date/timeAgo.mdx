 }

### props
| 属性名 | 说明 | 类型 | 必填 | 默认值 |
| --- | --- | --- | --- | --- |
| *timestamp* | *时间戳* | *Number* | *✅* | *-* |
| *option* | *配置项* | *Object* | *❌* | *-* |
| *option.open* | *自定义* | *Boolean* | *✅* | *-* |
| *option.cb* | *回调* | *Function* | *✅* | *-* |
{/* | *format* | *格式* | *String* | *❌* | *hh:mm* | */}
```javascript
interface TimeUnits {
  seconds: number;
  minutes: number;
  hours: number;
  days: number;
  months: number;
  years: number;
}
interface Options {
  open:boolean,
  cb:(e:TimeUnits) => string
}
```
