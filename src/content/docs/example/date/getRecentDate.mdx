<CodeRunner code={`
const data = zztool.getRecentDate(new Date());
console.log(data)
const data1 = zztool.getRecentDate(new Date(), 5, "after", "Y-M-D", { 1: 10, 2: 70, 3: 150, 4: 300,5: 11 });
console.log(data1)
`} />

### props
| 属性名 | 说明 | 类型 | 必填 | 默认值 |
| --- | --- | --- | --- | --- |
| *date* | *日期* | *Date \| String \| Number* | *✅* | *-* |
| *step* | *步长* | *Number* | *❌* | *1 \| 2 \| 3 \| 4* |
| *type* | *类型* | *String* | *❌* | *before \| after* |
| *format* | *格式化格式* | *String* | *❌* | *Y-M-D* |
| *option* | *步长对应天数* | *Record\<number,number\>* | *❌* | *\{ 1: 3, 2: 7, 3: 15, 4: 30 \}* |

返回值
