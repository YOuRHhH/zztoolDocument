### props
| 属性名 | 说明 | 类型 | 必填 | 默认值 |
| --- | --- | --- | --- | --- |
| *data* | *数据* | *Function* | *✅* | *-* |
| *type* | *类型* | *Enum* | *✅* | *url* |

```java
interface type { 
  url: string,
  json: string,
  formData: string
}
```
